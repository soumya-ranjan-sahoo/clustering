About

ClusterUI:a web tool for clustering multivariate data

This is an unsupervised learning web tool that clusters microstructures in steel based on their physical properties. You can upload your own data as an Excel or CSV file and get a summary of it under the Upload tab. Different clustering algorithms can be selected under the Cluster tab. You can reduce the dimensionality of the data by selecting Principal Component Analysis (PCA).

This web tool is developed as part of the coursework - Software Engineering, 2019 for the Chair for Functional Materials, Saarland University.

More information on the department can be found on their webpages: https://www.fuwe.uni-saarland.de/
